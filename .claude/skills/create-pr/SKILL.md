name: create-pr
description: Create pull request for a feature or bug fix.
---
Create a pull request for the following feature or bug fix:
- create branch from main
- commit changes with a descriptive message
- push branch to remote
- open pull request with a clear title and description
- add relevant reviewers, usually the people who worked on the related code and are active in the repository
- add labels (bug or dependency update if tag available)
- link any related issues or pull requests
